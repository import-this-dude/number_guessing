# 🔢 Number Guessing Game

A fun and beginner-friendly Number Guessing Game built with Python.

In this game, the computer randomly selects a number, and the player must guess the correct number within a limited number of attempts.

This project is great for practicing Python basics such as loops, conditions, user input, and random number generation.

---

## 🎮 Features

- Random number generation
- User input handling
- Hint system (Too High / Too Low)
- Win and lose conditions
- Beginner-friendly terminal interface

---

## 🛠 Built With

- Python 3
- random module

---

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/your-username/number-guessing-game.git
2. Open the project folder
cd number-guessing-game
3. Run the game
python number_guessing_game.py

🎲 How to Play

* The computer chooses a random number.
* Enter your guess in the terminal.
* The game will tell you if your guess is:
    * Too High 📈
    * Too Low 📉
* Keep guessing until you find the correct number.
